# **Shop Management Application**

### **Описание**
Shop Management Application — это приложение, которое моделирует работу магазина. Оно реализовано на языке C# с использованием .NET SDK и Entity Framework Core. Приложение позволяет управлять товарами и магазинами, а также выполнять различные операции, связанные с покупками, поиском товаров и анализом данных.

---

## **Функционал**
Приложение поддерживает следующие операции:
1. **Создать магазин:** Добавить новый магазин с уникальным кодом, названием и адресом.
2. **Создать товар:** Добавить новый товар с уникальным названием.
3. **Завезти товар в магазин:** Установить цену и количество для товара в конкретном магазине.
4. **Найти магазин с самым дешевым товаром:** Определить, в каком магазине заданный товар стоит дешевле всего.
5. **Узнать, что можно купить на сумму:** Показать список товаров, которые можно купить в магазине на заданную сумму.
6. **Купить товары в магазине:** Покупка партии товаров в магазине, с проверкой наличия.
7. **Найти магазин с минимальной стоимостью партии:** Определить, в каком магазине партия товаров (набор) стоит дешевле всего.
8. **Показать все магазины:** Просмотреть список всех магазинов.
9. **Показать все товары:** Просмотреть список всех товаров.

---

## **Архитектура проекта**
Проект разделен на три слоя:
1. **DAL (Data Access Layer):**
   - Работа с базой данных (SQLite) через Entity Framework Core.
   - Интерфейсы и реализации репозиториев (`IShopRepository`, `IProductRepository`).
2. **BLL (Business Logic Layer):**
   - Основная бизнес-логика приложения.
   - Сервисный класс `ShopService` для выполнения операций.
3. **UI (User Interface):**
   - Консольное меню для взаимодействия с пользователем.
   - Обработка пользовательского ввода и вызовы методов сервиса.

---

## **Стек технологий**
- **Язык:** C#
- **Фреймворк:** .NET SDK
- **База данных:** SQLite
- **ORM:** Entity Framework Core
- **Инструменты миграции:** EF Core CLI
- **Формат ввода/вывода:** Консольное приложение
